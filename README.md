## 📖 About The Project

**One Nation One Portal** is a centralized digital platform designed to simplify access to government services, welfare schemes, public resources, and official announcements through a single user-friendly interface.

The platform aims to bridge the gap between citizens and government services by providing an organized, accessible, and modern digital experience.

Instead of navigating multiple government websites, users can access important information and services from one unified portal.

---

## 🎯 Objectives

* Simplify citizen access to government services
* Improve awareness of government schemes and initiatives
* Provide a centralized information platform
* Enhance accessibility through a responsive design
* Deliver a modern and intuitive user experience

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/thanishkaykb/One-Nation-One-Portal.git
```

### Navigate to Project Directory

```bash
cd One-Nation-One-Portal
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
